Boost QVM

scalar

#include <boost/qvm/s_traits.hpp>

namespace boost
{
    namespace qvm
    {
        template <class T>
        struct scalar
        {
            typedef /*exact definition unspecified*/ type;
        };
    }
}

The expression v_traits<T>::scalar_type evaluates to the scalar type of the vector type T (if is_v<T>::value is true).

The expression m_traits<T>::scalar_type evaluates to the scalar type of the matrix type T (if is_m<T>::value is true).

The expression q_traits<T>::scalar_type evaluates to the scalar type of the quaternion type T (if is_q<T>::value is true).

The expression scalar<T>::scalar_type is similar, except that it automatically detects whether T is a vector or a matrix or a quaternion type.
